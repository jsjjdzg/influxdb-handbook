## 使用方法

如果嫌在线速度慢的话，也直接在本地生成。具体方法如下：

1 安装NodeJS

2 安装gitbook

```bash
npm install gitbook -g
```

3 clone本仓库

4 生成html代码

```bash
gitbook build 
```

然后就可以直接在本地看啦
